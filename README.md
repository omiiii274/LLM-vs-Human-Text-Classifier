

AI-Generated vs. Human-Generated Text Classifier
MetBrains Data Science & AI Capstone Project

🚀 Project Overview
This project involves building an end-to-end NLP pipeline designed to distinguish between text authored by humans and content generated by Large Language Models (LLMs) like GPT, Claude, or Gemini. By analyzing linguistic "fingerprints," the system identifies structural patterns unique to AI writing styles.

💼 Key Business Use Cases
EdTech: Identifying AI-generated academic submissions to maintain academic integrity.

E-commerce: Detecting bot-written fake product reviews to ensure consumer trust.

Media: Identifying AI-generated social media content to combat automated misinformation and fake news.

🛠 Technical Stack
Language: Python.

NLP Libraries: spaCy (Lemmatization), Scikit-learn (TF-IDF), Transformers (DistilBERT).

Analysis & Visualization: textstat (Flesch Reading Ease), Matplotlib, Seaborn, WordCloud.

Model Explainability: SHAP and LIME (to highlight influential keywords).

Modeling: Random Forest, XGBoost, and Transformers.

Deployment: Streamlit.

🧬 Deliverables (8-Stage Pipeline)
Problem Understanding: Defining 3 real-world business applications.

Preprocessing: spaCy-based cleaning and lemmatization to focus on "lemmas" and structural intent.

EDA: Analysis of text length distribution, Flesch Reading Ease, and N-grams.

Feature Engineering: Implementation of TF-IDF and Word Embeddings.

Modeling: Comparison of classical ML (Random Forest, XGBoost) and Deep Learning (Transformers).

Evaluation: Metrics including Precision, Recall, and Confusion Matrices.

Explainability: Using SHAP/LIME for model transparency.

Deployment: Interactive web application built with Streamlit.

⚙️ How to Run
Install dependencies:

Bash
pip install spacy textstat shap lime streamlit xgboost transformers scikit-learn pandas matplotlib seaborn wordcloud
Download the spaCy NLP model:

Bash
python -m spacy download en_core_web_sm
Run the application:

Bash
streamlit run app.py
🧠 NLP Pipeline Detail
Standard cleaning can sometimes remove the "human" element. This pipeline focuses on lemmas to capture structural intent:

Basic Cleaning: Lowercasing and whitespace removal.

Noise Removal: Stripping punctuation while keeping word context.

spaCy Lemmatization: Converting words to their root form and filtering stop words to identify unique writing styles.
